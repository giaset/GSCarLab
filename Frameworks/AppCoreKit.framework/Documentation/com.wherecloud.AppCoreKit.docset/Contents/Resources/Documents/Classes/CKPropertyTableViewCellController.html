Inherits from CKTableViewCellController : CKCollectionCellController : NSObject
Declared in CKPropertyTableViewCellController.h

Overview

CKPropertyTableViewCellController is a base class for cell controllers working with a CKProperty value. This provides the base value for the mecanisms that should be implemented in subclasses like readonly, or value validation.
